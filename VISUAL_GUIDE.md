## Color Coding

- **White Background:** Main content sections
- **Gray Background:** Alternating sections for visual rhythm
- **Dark Blue Gradient:** Achievements (premium feel)
- **Orange Gradient:** Registration CTA (high energy)
- **Dark Background:** Footer (grounding)

## Interaction Points

✅ **Clickable:**
- Navbar links
- Hero slider arrows & dots
- Faculty carousel arrows & dots
- Gallery images (opens lightbox)
- All buttons and CTAs
- Footer links
- Social media icons

✅ **Hoverable:**
- All cards scale/shadow
- Images scale
- Links underline
- Buttons elevate

✅ **Responsive:**
- Navbar → Hamburger menu (mobile)
- 4 cols → 2 cols → 1 col (cards)
- Side-by-side → Stacked (sections)

## Typography Hierarchy

```
H1 (Hero): 48-60px Poppins Bold
H2 (Sections): 36-48px Poppins Bold
H3 (Cards): 24-32px Poppins Semibold
Body: 16px Inter Regular
Small: 14px Inter Regular
Caption: 12px Inter Medium
```

## Spacing System

```
Section Padding: 64-96px vertical
Card Padding: 24px
Gap between cards: 24-32px
Container max-width: 1280px
Container padding: 16-32px horizontal
```

This creates a professional, balanced, and visually appealing website! 🎨
