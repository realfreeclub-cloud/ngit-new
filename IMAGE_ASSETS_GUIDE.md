# Image Assets Guide for NGIT Website

## Required Images

### Hero Slider Images (1920x1080px)
Place these in `/public/images/`:

1. **hero-1.jpg** - Main campus or students studying
   - Suggested: Group of students in modern classroom
   - Theme: Learning, focus, success

2. **hero-2.jpg** - Results celebration
   - Suggested: Students celebrating with certificates
   - Theme: Achievement, joy, success

3. **hero-3.jpg** - Infrastructure
   - Suggested: Modern AC classroom or library
   - Theme: Facilities, technology, comfort

### About Section Images
4. **about-1.jpg** (800x1000px) - Campus exterior or entrance
5. **about-2.jpg** (800x450px) - Classroom scene
6. **about-3.jpg** (800x450px) - Library or lab
7. **about-4.jpg** (800x1000px) - Faculty teaching

## How to Add Images

1. Create `/public/images/` directory if it doesn't exist
2. Add your images with the exact names listed above
3. Images will automatically load in the components

## Image Optimization Tips

- Use WebP format for better compression
- Optimize images before uploading (use tools like TinyPNG)
- Recommended dimensions are listed above
- Keep file sizes under 500KB each

## Using Unsplash (Free Stock Photos)

Search for:
- "indian students studying"
- "classroom modern"
- "educational institute"
- "students celebrating"
- "library study"

Download high-resolution images and rename them according to the list above.
